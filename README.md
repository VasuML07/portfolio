# Vasu Margana — Portfolio

A modern, dark-themed developer portfolio built with **Next.js 16**, **Tailwind CSS 4**, **Framer Motion**, and **shadcn/ui**.

![Next.js](https://img.shields.io/badge/Next.js-16-black?logo=next.js)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue?logo=typescript)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4-38bdf8?logo=tailwindcss)

## Features

- Dark minimalist design with blue accents
- Animated scroll reveals with Framer Motion
- Live **GitHub** stats (repos, stars, forks, languages, commit activity)
- Live **LeetCode** stats (problems solved, ranking, contest rating, acceptance rate)
- Responsive design — mobile-first
- Custom scrollbar & grid overlay background
- Subtle noise texture background
- Smooth section scrolling

## Sections

1. **Hero** — Name, title, availability badge, CTA buttons
2. **About** — Bio + code-block style config display
3. **Projects** — Bento grid with tags, GitHub links, live demos
4. **Education** — Academic details with focus areas
5. **Skills** — Category cards (AI/ML, Programming, Frameworks, Core)
6. **Coding Profiles** — Live GitHub & LeetCode stats
7. **Contact** — Social links, email, resume

## Getting Started

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build

# Start production server
bun run start
```

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import the repository
4. Click Deploy

That's it — no additional configuration needed!
